# Stax & Wax Landing Page

A modern, responsive landing page concept for a boutique vinyl pressing house. This project focuses on high-fidelity UI design, sophisticated typography, and smooth, tactile CSS animations to mirror the craftsmanship of the vinyl experience.

## Features
- **Minimalist Design:** Clean typography and a curated color palette to highlight brand storytelling.
- **Interactive UI:** Features custom-animated components, including a rotating vinyl record, to provide engaging visual feedback.
- **Responsive Architecture:** Built with a mobile-first approach to ensure a premium user experience across all devices.
- **Catalog & Requests:** Integrated "Now Spinning" and "Full Catalog" views, along with a functional B-Side request form for custom inquiries.

## Tech Stack
- HTML5
- CSS3 (Animations/Transitions)
